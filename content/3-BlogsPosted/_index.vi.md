### [Blog 1 - CYBER SECURITY TRÊN AWS: KHÔNG CHỈ DÙNG WAF, HÃY TỰ ĐỘNG HÓA WAF](3.1-Blog1/)
Blog này giới thiệu giải pháp Security Automations for AWS WAF, giúp tự động triển khai và vận hành nhiều lớp bảo vệ như AWS Managed Rules, SQL Injection/XSS protection, HTTP Flood protection, Bad Bot Detection, Threat Intelligence và log-driven protection. Đây là hướng tiếp cận giúp WAF không chỉ là tập hợp rule tĩnh, mà trở thành một cơ chế phòng thủ có khả năng quan sát, phân tích và phản ứng liên tục.

### [Blog 2 - CYBER SECURITY TRÊN AWS: TỪ WAF TỰ ĐỘNG HÓA ĐẾN HỆ THỐNG PHÁT HIỆN XÂM NHẬP BẰNG MACHINE LEARNING](3.2-Blog2/)
Blog này chia sẻ hướng nghiên cứu xây dựng Network Intrusion Detection System (NIDS) trên AWS sử dụng Machine Learning với bộ dữ liệu CSE-CIC-IDS2018 on AWS. Dự án tập trung vào việc học từ dữ liệu lưu lượng mạng để phát hiện hành vi bất thường, kết hợp giữa rule-based detection và machine learning-based detection nhằm nâng cao khả năng phát hiện các mối đe dọa mới.

### [Blog 3 - MINECRAFT CYBER SECURITY TRÊN AWS: KHÔNG CHỈ LÀ MỞ SECURITY GROUP, HÃY TỰ ĐỘNG HÓA PHẢN ỨNG SỰ CỐ](3.3-Blog3/)
Blog này trình bày kiến trúc bảo vệ Minecraft Server trên AWS theo hướng SOAR cho game server, kết hợp AWS Global Accelerator, AWS Shield, Network Load Balancer, Systems Manager Session Manager, GuardDuty, EventBridge, Lambda, Network ACL, SNS và AWS Backup. Giải pháp giúp ẩn IP gốc, phát hiện tấn công, tự động cô lập hacker và đảm bảo khả năng khôi phục dữ liệu khi xảy ra sự cố.
