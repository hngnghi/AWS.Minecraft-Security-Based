### [Blog 1 - CYBER SECURITY ON AWS: DON'T JUST USE WAF — AUTOMATE IT](3.1-Blog1/)
This blog introduces **Security Automations for AWS WAF**, an AWS solution that automates the deployment and operation of multiple security layers, including AWS Managed Rules, SQL Injection/XSS protection, HTTP Flood protection, Bad Bot Detection, Threat Intelligence integration, and log-driven protection. Rather than treating WAF as a static collection of rules, the solution demonstrates how to build a continuously adaptive defense mechanism through monitoring, analysis, and automated response.

### [Blog 2 - CYBER SECURITY ON AWS: FROM AUTOMATED WAF TO A MACHINE LEARNING-BASED NETWORK INTRUSION DETECTION SYSTEM](3.2-Blog2/)
This blog explores the development of a **Machine Learning-based Network Intrusion Detection System (NIDS)** on AWS using the **CSE-CIC-IDS2018 on AWS** dataset. The project focuses on learning from network traffic to distinguish normal and malicious behavior, combining traditional rule-based detection with machine learning techniques to improve the detection of previously unseen cyber threats.

### [Blog 3 - MINECRAFT CYBER SECURITY ON AWS: DON'T JUST OPEN SECURITY GROUPS — AUTOMATE INCIDENT RESPONSE (SOAR FOR GAME SERVERS)](3.3-Blog3/)
This blog presents a **SOAR-inspired security architecture** for protecting a Minecraft server on AWS. The solution integrates AWS Global Accelerator, AWS Shield, Network Load Balancer, Systems Manager Session Manager, Amazon GuardDuty, EventBridge, AWS Lambda, Network ACLs, Amazon SNS, and AWS Backup to hide the origin IP, detect attacks in real time, automatically isolate attackers, and ensure cyber resilience through automated backup and disaster recovery.
