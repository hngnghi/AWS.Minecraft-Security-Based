# 1. Purpose of the Event

The event was organized as an academic knowledge-sharing activity for students participating in the AWS program. Its purpose was to help participants review and strengthen their understanding of Cloud Computing, AWS services, and practical cloud-related concepts.

In addition, the event encouraged participants to improve their critical thinking, quick response ability, and teamwork skills through a knowledge competition format. It also created an opportunity for students to interact, exchange experiences, and gain motivation to continue learning and developing their skills in Cloud Computing.

---

# 2. Speakers and Organizers

The event involved the following groups:

- The organizing team of **The First Cloud Journey – AWS Vietnam**.
- Mentors and trainers from the AWS program.
- Judges responsible for evaluating and managing the competition.

---

# 3. Event Highlights

The highlight of the program was a knowledge competition among **8 teams**.

The teams answered questions related to:

- Fundamental knowledge of Cloud Computing.
- Popular AWS services such as EC2, S3, IAM, VPC, RDS, and others.
- Security on AWS.
- Networking concepts.
- Best practices for deploying cloud systems.
- Practical scenarios requiring participants to choose the correct solutions.

After each round, teams with higher scores advanced to the next stage. Finally, the best-performing teams competed in the final round to determine the winners.

The event atmosphere was exciting and energetic. All teams demonstrated strong knowledge retention, quick thinking, and excellent teamwork.

---

# 4. Knowledge Gained

Through the competition, I learned and reviewed many useful topics, including:

- AWS knowledge covered during the program.
- How AWS services can be applied in real-world scenarios.
- Useful tips and best practices for cloud deployment.
- How to analyze technical questions and make decisions within a limited time.

---

# 5. Practical Applications

The knowledge gained from the event can be applied to:

- Deploying virtual servers using Amazon EC2.
- Managing access permissions with AWS IAM.
- Designing network architectures using Amazon VPC.
- Storing data with Amazon S3.
- Building secure and cost-optimized cloud systems.
- Preparing for future AWS certification exams.

# 6. Event Experience

## Learning from Experienced Mentors

The mentors shared practical experience in using AWS services, explained questions from the competition, and provided useful advice. Their explanations helped participants better understand how to deploy services in cloud environments.

## Practical Technical Experience

Although the program was organized as a competition, each question was closely connected to real-world AWS use cases.

Through the event, I gained a better understanding of:

- Cloud architecture design.
- System security.
- Service scalability.
- AWS resource management.

## Exposure to Modern Tools

The event helped participants become more familiar with popular AWS services, including:

- Amazon EC2
- Amazon S3
- IAM
- VPC
- RDS
- CloudWatch

It also helped me understand how these services can be combined to build a complete cloud-based system.

## Networking and Knowledge Exchange

Through the team competition activities, I had the opportunity to:

- Communicate with other participants.
- Exchange AWS knowledge.
- Learn different problem-solving approaches from other teams.
- Expand my network within the Cloud Computing community.

# 7. Lessons Learned

After participating in the event, I learned several important lessons:

- It is important to master the fundamentals of AWS rather than only learning theoretical concepts.
- Teamwork and idea exchange are essential not only in competitions but also in real-world work environments.
- Regular hands-on practice on AWS helps improve long-term knowledge retention.
- Knowledge competitions are valuable opportunities to evaluate personal abilities and identify areas that need further improvement.
