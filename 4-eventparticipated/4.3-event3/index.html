Event 3

AWS MEETUP EVENT REPORT

InformationDetails
Event NameFirst Cloud AI Journey – Agentic AI Build Week
Date & Time09:00 AM, May 30, 2026
Venue26th Floor, Bitexco Financial Tower, 02 Hai Trieu Street, Sai Gon Ward, Ho Chi Minh City, Vietnam
RoleAttendees

1. Purpose of the Event

The First Cloud AI Journey – Agentic AI Build Week was organized to introduce participants to practical learning pathways in Cloud Computing and Artificial Intelligence. The event focused on AWS technologies, cloud cost optimization strategies, and the growing potential of Agentic AI systems across various real-world applications.

In addition to technical knowledge, the event created opportunities for students, AI developers, industry professionals, and technology companies to connect, exchange ideas, and build a collaborative learning community.

2. Organizers and Participants

The event was organized by:

  • GenAI Fund
  • First Cloud AI Journey
  • Technology partners:
    • AWS
    • NVIDIA
    • Google Cloud

Participants included:

  • More than 2,000 AI Builders and software developers.
  • Approximately 30 supporting organizations.
  • Representatives from 10 technology companies.
  • Students, software engineers, and individuals interested in Cloud Computing and Artificial Intelligence.

3. Event Highlights

The event featured a variety of presentations covering personal development, Cloud Computing, and Artificial Intelligence.

Personal Development and Mindset

The speakers shared valuable insights into self-improvement and professional growth, including:

  • The Iceberg of Procrastination: Understanding that procrastination is often driven by psychological barriers such as fear of failure and fear of being judged rather than laziness.
  • Confidence: Emphasizing the importance of confidence in communication, networking, and career development, along with practical methods for building self-confidence through action.

AWS Cloud

The technical sessions introduced a structured learning roadmap for AWS:

  • Building cloud architecture fundamentals through AWS Cloud Quest.
  • Developing and testing applications locally using LocalStack.
  • Deploying real-world applications on AWS Cloud.

LocalStack was introduced as an efficient solution for simulating AWS services on a local machine, allowing developers to learn and test cloud architectures without incurring AWS costs.

Featured Technology Projects

Several innovative AI projects were presented during the event, including:

  • Tu Vi Dai Viet: An AI-powered application that modernizes traditional Vietnamese astrology by making ancient concepts easier for younger generations to understand.
  • SynthHunter: An advanced AI-based voice authentication and deepfake detection system utilizing Speech Dynamics, Encoder Behavior, and Temporal Rhythm to improve fraud prevention.

4. Knowledge Gained

After attending the event, I gained valuable knowledge and practical insights, including:

  • A clearer learning roadmap for AWS Cloud technologies.
  • Practical experience using LocalStack for AWS development and testing.
  • A better understanding of Agentic AI and its real-world applications.
  • The importance of building a strong personal portfolio through hands-on projects.
  • The value of confidence, continuous learning, and the “Learn by Doing” mindset.

5. Practical Applications

The knowledge gained from the event can be applied to:

  • Practicing AWS services using AWS Cloud Quest and LocalStack.
  • Building cloud-based applications for coursework and personal projects while minimizing cloud costs.
  • Developing AI applications on cloud platforms.
  • Monitoring AWS Billing and managing cloud resources efficiently.
  • Building a personal GitHub portfolio through practical projects.
  • Preparing for Hackathons and future AWS certification exams.

6. Event Experience

Learning from Industry Speakers

The speakers shared practical experiences from developing AI products and deploying cloud solutions on AWS. Their presentations combined technical expertise with valuable career development advice and effective learning strategies.

Exposure to Modern Technologies

The event introduced participants to a wide range of modern technologies, including:

  • AWS Cloud
  • AWS Cloud Quest
  • LocalStack
  • Agentic AI
  • Amazon Bedrock
  • Large Language Models (LLMs)
  • AI Applications
  • Cloud Architecture

These sessions demonstrated how modern AI technologies can be integrated with cloud infrastructure to build scalable and intelligent applications.

Practical Learning Experience

Beyond theoretical discussions, the event featured numerous real-world case studies and demonstrations that helped participants better understand:

  • AI product development processes.
  • Cloud cost management.
  • Cloud architecture design.
  • The “Learn by Doing” approach to mastering technology.

Networking Opportunities

The event also provided opportunities to:

  • Connect with AI Builders and software engineers.
  • Engage with the AWS and AI communities.
  • Exchange learning experiences and career advice.
  • Expand professional networks with technology companies and industry experts.

7. Lessons Learned

After attending the event, I learned several valuable lessons:

  • The most effective way to master technology is by combining theoretical knowledge with hands-on practice.
  • Building a strong portfolio through real-world projects is essential for career development.
  • Cloud cost management is an important skill for every AWS practitioner.
  • Taking action consistently helps build confidence and improve technical skills.
  • Participating in technology communities, conferences, and Hackathons provides excellent opportunities for continuous learning, networking, and professional growth.

8. Conclusion

The First Cloud AI Journey – Agentic AI Build Week was a highly valuable learning experience that provided practical knowledge in AWS Cloud, Agentic AI, and modern software development practices. The event not only expanded my technical knowledge but also inspired me to adopt a more proactive learning mindset through building real-world projects. It has strengthened my motivation to continue developing my skills and preparing for future opportunities in Cloud Computing and Artificial Intelligence.
